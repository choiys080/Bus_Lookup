# B. Braun Korea 액티비티 포털 - 운영 가이드

## 개요

이 시스템은 B. Braun Korea 행사 참가자들이 QR 코드를 스캔하여 자신의 액티비티 일정을 확인할 수 있는 웹 포털입니다.

## 관리자 접근 방법

1. 웹사이트 접속
2. 우측 상단 **지도 아이콘** 클릭
3. 비밀번호 입력: `admin123`
4. 관리자 화면에서 CSV 업로드 가능

## 참가자 데이터 업로드

### CSV 파일 형식

| 열 이름 | 설명 | 필수 |
|---------|------|------|
| 이름 | 참가자 성명 | ✓ |
| 부서 | 소속 부서 | |
| 휴대전화 | 연락처 (형식 무관) | ✓ |
| 액티비티 | 투어 프로그램명 | |
| 출발시간 | 집합 시간 | |
| 집합장소 | 집합 위치 | |
| 가이드 정보 | 담당 가이드 | |
| 일정 1~3 | 세부 일정 | |
| 준비물 | 필요 물품 | |
| 주의사항 | 안내 메시지 | |

### 업로드 절차

1. Excel에서 데이터 준비
2. **CSV (UTF-8)** 형식으로 저장
3. 관리자 모드에서 파일 업로드
4. "✓ 동기화 완료" 메시지 확인

## 참가자 사용 방법

1. QR 코드 스캔 → 웹사이트 접속
2. 이름과 휴대전화 번호 입력
3. "내 일정 확인하기" 버튼 클릭
4. 개인 일정 확인

> **참고**: 전화번호는 형식에 관계없이 입력 가능합니다.
> 예: `010-1234-5678` 또는 `01012345678`

## 체크인 기록

관리자 화면에서 참가자 체크인 로그를 확인하고 CSV로 내보낼 수 있습니다.

## 문제 해결

| 문제 | 해결 방법 |
|------|----------|
| "정보를 찾을 수 없습니다" | 전화번호 재확인, 안내데스크 문의 |
| CSV 업로드 실패 | UTF-8 인코딩 확인, 필수 열 존재 확인 |
